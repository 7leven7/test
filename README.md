## Installation

sudo docker-compose up , <br>
sudo docker exec -it  citrus_php_1 bash , <br>
cd /code ,<br>
composer update .


In the config.php file you can change database credentials.<br>
For database you can find the citrus.sql file in the assets folder.<br>
